# 📊 Online Feedback Collector with Admin Dashboard

## 📌 Description

This project is a web-based application that allows users to submit feedback online and enables the admin to view and manage all feedback through a dashboard.

---

## 🎯 Features

* User feedback form (Name, Email, Message, Rating)
* Admin login system
* View all feedback in one place
* Delete/manage feedback
* Simple and user-friendly interface

---

## 🛠️ Technologies Used

* HTML, CSS, JavaScript
* Python (Flask) / PHP
* MySQL / SQLite

---

## ⚙️ Installation

```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
pip install -r requirements.txt
python app.py
```

---

## ▶️ Usage

1. Open the application in browser
2. Submit feedback as a user
3. Login as admin to view dashboard
